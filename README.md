# DuskServices

Landing page for Dusk Services game lobby UI.

## Pages
- `index.html` – Home
- `unlock.html` – Unlock All lobby ($10)
- `level1001.html` – Level 1001 lobby ($15)

## Deployment
1. Commit and push to GitHub.
2. GitHub Pages: Settings → Pages → Branch `main` → Root `/`.
3. Wait, then open: `https://<username>.github.io/<repo>/`.
